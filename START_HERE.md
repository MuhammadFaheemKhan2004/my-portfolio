## ⚡ Quick Start (3 Steps)

### Step 1: Run Setup
```bash
# Windows
setup.bat

# Mac/Linux
chmod +x setup.sh && ./setup.sh
```

### Step 2: Configure Backend
Edit `server/.env`:
```env
MONGODB_URI=mongodb://localhost:27017/portfolio
EMAIL_USER=your-email@gmail.com
EMAIL_PASSWORD=your-app-password
```

### Step 3: Start
```bash
npm run dev
```

**Access:** http://localhost:5173
